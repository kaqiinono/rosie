### 23-P2 · 课前测2 · 九张牌积与和

- 课程：第23讲·逻辑推理（第23讲）
- 分区：课前测
- 题型：题型5·复合逻辑
- 难度：4 星
- 应用链接：/math/ny/23/pretest/2

**题目**

老师手里有九张卡片，上面分别写着 1～9 共九个数字。明明、芳芳和亮亮每人拿了三张。
明明说："我的三张卡片数字之积是 **48**。"
芳芳说："我的三张卡片数字之和是 **15**。"
亮亮说："我的三张卡片数字之积是 **63**。"
明明的三张卡片中最大的数字是多少？

**问题**：明明三张牌中最大的数字是几？（）

**解析**

- 亮亮积63 = 7×9×1 → 亮亮拿 {1,7,9}
- 从剩余 {2,3,4,5,6,8} 中，明明积48 = 2×3×8 → 明明拿 {2,3,8}
- 芳芳拿剩余 {4,5,6}，和 = 15 ✓
- 明明三张中最大的是 **8**

**答案**：8

### 23-P3 · 课前测3 · 谁做了好事

- 课程：第23讲·逻辑推理（第23讲）
- 分区：课前测
- 题型：题型3·假设验证法
- 难度：3 星
- 应用链接：/math/ny/23/pretest/3

**题目**

甲、乙、丙三人中有一人做了一件好事，他们各自都说了一句话，而且其中只有一句真话。
甲说："是乙做的。"
乙说："不是我做的。"
丙说："也不是我做的。"
那么到底是谁做的好事？

**问题**：是谁做了好事？（1=甲/2=乙/3=丙）（）

**解析**

- 假设是丙做的：甲说"是乙"=假，乙说"不是我"=真，丙说"不是我"=假 → 恰好1句真话 ✓
- 假设是乙做的：甲=真，乙=假，丙=真 → 2句真话 ✗
- 假设是甲做的：甲=假，乙=真，丙=真 → 2句真话 ✗
- 所以是**丙**做了好事

**答案**：3

### 23-P4 · 课前测4 · 各说对一半

- 课程：第23讲·逻辑推理（第23讲）
- 分区：课前测
- 题型：题型4·双半真
- 难度：3 星
- 应用链接：/math/ny/23/pretest/4

**题目**

甲、乙、丙、丁四位同学的运动衫上印上了不同的号码。
赵说："甲是2号，乙是3号。"
钱说："丙是4号，乙是2号。"
孙说："丁是2号，丙是3号。"
李说："丁是1号，乙是3号。"
又知道赵、钱、孙、李每人都只说对了一半。请问：**丙的号码是几号？**

**问题**：丙的号码是几号？（）

**解析**

- 假设乙=3（赵后半真）→ 赵前半假：甲≠2；钱后半乙=2假 → 钱前半丙=4真 → 丙=4
- 孙后半丙=3假 → 孙前半丁=2真 → 丁=2；李前半丁=1假 → 李后半乙=3真✓
- 甲≠2，丙=4，丁=2，乙=3 → 甲=1
- 验证：赵(甲2假乙3真)✓，钱(丙4真乙2假)✓，孙(丁2真丙3假)✓，李(丁1假乙3真)✓

**答案**：4

有四个小朋友悦悦、苗苗、风风和奇奇喜欢画鸡，每人各画了一只，有黑公鸡、白公鸡、黑母鸡、白母鸡。已知：
1）悦悦画的鸡正在打鸣；
2）苗苗和悦悦画的鸡都是黑色的；
3）风风和苗苗画的鸡都是母鸡。
那么四个小朋友分别画的是哪只鸡？

地理课上有 5 个省份分别编上了数字 1 至 5，请同学们猜一猜每个编号是哪个省。
立立说："2号是陕西，5号是江苏。"
苗苗说："2号是湖南，4号是山东。"
风风说："1号是山东，5号是四川。"
悦悦说："3号是湖南，4号是四川。"
奇奇说："2号是江苏，3号是陕西。"
这5名同学每人都只答对了一个省，并且每个编号只有一个人答对，那么2号是什么省？

### 23-L12 · 练一练 · 四兄弟排行

- 课程：第23讲·逻辑推理（第23讲）
- 分区：课堂
- 题型：题型5·复合逻辑
- 难度：4 星
- 应用链接：/math/ny/23/lesson/12

**题目**

有 A、B、C、D 四兄弟，已知：
①A 不是老大，也不是老四；
②B 不是老三，也不是老大；
③如果 A 不是老三，那么 C 不是老大；
④D 不是老四，也不是老大。
请问谁是老大？

**问题**：谁是老大？（1=A/2=B/3=C/4=D）（）

**解析**

- 重新：D:1黄假(4=黄) → D5=紫真 → 5=紫；B:5白假(5=紫)✓ → B2=红假✓(2=白)
- 结果：1=红，2=白，3=蓝，4=黄，5=紫；第3封是**蓝色**

**答案**：2

### 23-W11 · 闯关11 · 铅笔数量

- 课程：第23讲·逻辑推理（第23讲）
- 分区：练习册
- 题型：题型4·双半真
- 难度：4 星
- 应用链接：/math/ny/23/workbook/11

**题目**

纳纳、约约、晨晨、牛牛收到了不同数量的铅笔作为奖品。
小东说："纳纳有2支，约约有3支。"
小南说："约约有4支，晨晨有2支。"
小西说："牛牛有2支，晨晨有3支。"
小北说："牛牛有1支，约约有3支。"
又知道小东、小南、小西、小北每人都只说对了一半，请问晨晨收到了多少支铅笔？

**问题**：晨晨收到了多少支铅笔？（支）

**解析**

- 假设约约=4（小南前半真）→ 小南晨晨2假；小东约3假 → 小东纳2真 → 纳=2
- 小北约3假 → 小北牛1真 → 牛=1；小西牛2假(牛=1) → 小西晨3真 → 晨=3
- 验证：纳=2，约=4，晨=3，牛=1（各不同✓）
- 晨晨收到 **3** 支铅笔

**答案**：3 支

### 23-W12 · 闯关12 · 挑选学校

- 课程：第23讲·逻辑推理（第23讲）
- 分区：练习册
- 题型：题型5·复合逻辑
- 难度：5 星
- 应用链接：/math/ny/23/workbook/12

**题目**

从奋斗、复兴、纳约、前进、光明五所小学中挑选交流学校：
①若去奋斗，则必须去复兴；
②复兴、纳约两所中至多去一所；
③前进、光明两所中至少去一所；
④纳约、前进两所都去或都不去；
⑤若去光明，一定要去奋斗和前进。
最终去了几所学校？

**问题**：最终去了几所学校？（所）

**解析**

- 假设去光明 → 条件⑤：去奋斗+前进 → 条件①：去复兴 → 条件②：有复兴则纳约不去 → 前进去了则纳约去（条件④）→ 矛盾
- 所以不去光明；条件③：前进、光明至少一所 → 去前进
- 条件④：前进去 → 纳约去；条件②：纳约去 → 复兴不去 → 条件①逆否：不去复兴则不去奋斗
- 选择：前进✓ 纳约✓，共 **2** 所

**答案**：2 所
