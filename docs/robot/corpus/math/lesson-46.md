- 题型：保证两个相同
- 难度：2 星
- 应用链接：/math/ny/46/pretest/1

**题目**

有 A、B、C、D、E **五种**类型的练习本各 3 个，至少要取出多少个练习本才能保证存在 **两个同类型**的练习本？

**问题**：至少要取多少个？（个）

**解析**

- 最不利：5 种各取 1 个 = 5 个，仍然两两不同
- 再取 1 个必与已有某种相同
- 5 + 1 = **6** 个

**答案**：6 个
