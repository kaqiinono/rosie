- 课程：第30讲·和差倍进阶（第30讲）

- 题型：题型3·三量和倍
- 难度：3 星

**题目**

体育室买来 **75** 个球，其中篮球是足球的 **2** 倍，排球比足球多 **3** 个，这三种球各多少个？

**问题**：足球有多少个？（个）

**解析**

- 设足球 = x 个
- 篮球 = 2x，排球 = x + 3
- 足球 + 篮球 + 排球 = x + 2x + (x+3) = 4x + 3 = 75
- 4x = 72，x = 18，即足球 = **18** 个
- 篮球 = 36 个，排球 = 21 个

**答案**：18 个

**解析**

- 被减数 = 减数 + 差（减法关系）
- 设差 = d，则减数 = d + 10
- 被减数 = (d+10) + d = 2d + 10
- 总和：(2d+10) + (d+10) + d = 4d + 20 = 80
- 4d = 60，d = 15（差），减数 = 25
- 被减数 = 25 + 15 = **40**

**题目**

纳约小学发动全校同学给山区学生捐书，一(1)班、一(2)班、一(3)班三个班共捐书 **300** 本，一(1)班、一(2)班两个班捐书总数比一(3)班多 **60** 本，如果一(3)班拿出 **20** 本给一(2)班，则两个班捐书数目相等。请问三个班各捐了多少本书？

**问题**：一班捐了多少本？（本）

**解析**

- 一(1)+一(2) = 一(3)+60，代入总和：(一(3)+60)+一(3)=300
- 2×一(3) = 240，一(3) = 120，一(1)+一(2) = 180
- 一(3)拿20给一(2)后相等：一(3)−20 = 一(2)+20
- 120−20 = 100 = 一(2)+20，一(2) = 80
- 一(1) = 180 − 80 = **100**，一(2) = 80，一(3) = 120

**答案**：100 本

商店运来橘子、苹果、香蕉共 **53** 千克，橘子的重量比苹果的 **3** 倍少 **3** 千克，香蕉的重量比苹果的 **2** 倍多 **2** 千克，那么商店运来橘子多少千克？

### 30-W12 · 闯关12 · 甲班原有多少人

- 课程：第30讲·和差倍进阶（第30讲）
- 分区：练习册
- 题型：题型2·三量和差
- 难度：4 星

**题目**

纳约小学三年级有甲、乙、丙 **3** 个班，一共有学生 **72** 人，如果从甲班转出 **2** 个人到乙班，则甲、乙两班人数相同，如果这时再从丙班转出 **3** 个人到乙班，则乙、丙两班人数相同，请问甲班原有多少人？

**问题**：甲班原有多少人？（人）

**解析**

- 甲给乙2人后甲=乙：甲−2 = 乙+2 → 甲 = 乙+4 … ①
- 此时乙已变为乙+2，丙给乙3人后乙=丙：乙+2 = 丙−3 → 丙 = 乙+5 … ②

- 3乙 = 63，乙 = 21
- 甲 = 21+4 = **25**，丙 = 21+5 = 26

**答案**：25 人
