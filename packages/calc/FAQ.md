# 口算怎么练？——给家长的说明

用大白话讲清楚：口算模块在帮孩子练什么、题目怎么来、怎样才算「会了」。

---

## 一句话

**口算会多练孩子还不熟、算得慢的题；已经又快又对的题会少出现，偶尔再抽查一下。**

---

## 开始前：先选练什么

打开 **口算 → 设置**：

1. **勾选题型**  
   比如「20 以内进位加法」「×6、×7」「加减混合」等。只练勾上的内容。

2. **题量怎么定**  
   - **自动分配**：你定总题数（如 20 / 30），系统会往「更弱的题型」多给几道。  
   - **精准设置**：每种题型自己填要练几道。

3. **限时答题（可选）**  
   - 打开后：可编辑各题型的**目标时间**（建议秒数）。  
   - **关掉也可以**：每场开始前仍可选计时模式；宽松模式下按需显示倒计时，系统仍会按目标时间判断快慢。  
     （关掉只是不编辑目标时间，不是「不算慢」。）

4. **数字键盘答对即过（可选）**  
   默认开启：用数字键盘答题时，输入已与正确答案完全一致则自动提交、免点确认；答错仍须点确认。竖式、分数、余数不受影响。

5. **默认计时模式**  
   在设置里选好默认模式（宽松 / 严格 / 自定义加成）；每次开始练习前会预填，可当场改。  
   - **宽松**：超时不强制交卷，星星无加成（×1.0）。  
   - **严格**：倒计时到 0 算最终错误，星星 +20%（×1.2）。  
   - **自定义加成**：倒计时 = 目标时间 + 加成秒数，超时算最终错误；星星加成随加成秒数递减（每多 1 秒少 5 个百分点，最低 ×1.0）。  
   - 自定义模式下：倒计时**含**加成秒数，但「算得顺不顺」仍按**目标时间**判断——在目标与倒计时之间答对，可能仍记为卡顿。

设好后回首页点 **开始口算** 即可。

### 中后期提速题型（补充）

设置里还可勾选更细的计算梯度，例如：

- **100 以内凑整**（有限表）：优先练还没见过的补数。
- **两 / 三位数 × 一位数**：已拆成「不进位」与「进位」（三位数进位要求连续进位）。
- **两位数 ÷ 一位数（满十）**：与「多位数 ÷ 一位数」按是否需要中间余数分开练。
- **整十 / 整百乘法、除法**：练「剥零 → 口诀 → 回填零」。
- **整百 / 整千减多位数**：目前只练 `100−…` 与 `1000−…`。

## 一场练习里会发生什么

1. **出题**  
   按你选的题型出题；弱的、没见过的会多一点（见下文）。

2. **答题**  
   对了得星星；连对还有小奖励。  
   数字键盘题若开了「答对即过」，输满正确答案会自动提交；答错仍要点确认。  
   错了一般可以再试一次；还不对会记入错题，本场末尾还会再练一遍（**补练有上限**，见下文）。

3. **上场错题**  
   上一场还没「练会」的错题，可能混在本场里再出现（不是永远堆在最前面）。

4. **结束**  
   看本场正确率、用时；星星进钱包（严格/加成模式有场次星星加成），可去换奖券。

## 计时模式与补练规则

每场日常练习开始前可选三种计时模式（设置里可设默认值）：

| 模式 | 倒计时 | 到 0 时 | 星星加成 |
|------|--------|---------|----------|
| **宽松** | 有目标时间时可选显示 | 不强制交卷 | ×1.0 |
| **严格** | 始终显示目标时间 | 自动交卷，算最终错误 | ×1.2（+20%） |
| **自定义加成** | 目标时间 + 加成秒数 | 同严格 | ×max(1.0, 1.2 − 0.05×加成秒) |

- **目标时间**用于判断「算得顺不顺」（卡顿判定）；**倒计时**用于界面显示和严格/加成模式的自动交卷。自定义加成下，倒计时含加成秒数，但在目标时间之后答对仍可能记为卡顿。  
- 星星加成在**整场结束**时一次性结算到本场总星星（不是每题单独乘）。  
- 加成秒数示例：+0 秒 → ×1.2；+2 秒 → ×1.1；+4 秒及以上 → ×1.0。

**补练上限与单次清算**

- 主路径约 `N` 题；错题补练池上限 `M = max(3, floor(N × 15%))`。  
- 主路径做错或严格/加成超时 → 尽量入补练池；池满则本场不再补练该题（仍会记长期错题）。  
- 补练阶段每题**只给一次机会**，做完即结束，不再反复排队。

## 系统怎么决定「下一道出什么」

可以想成三个小原则：

### 1. 新题优先（九九表一类）

对 **2～9 的乘除**（有限的那张「表」）**以及 100 以内凑整**，系统会尽量先让孩子碰到**还没练过的算式**，少反复刷已经见过的。

大范围题（如万以内加减、很多混合题）没法一张表列完，就先多练、多记，等记得多了再按强弱分配。

### 2. 算对但太慢 = 还没真正会

口算目标不只是「算对」，还要「算得顺」。

- 在建议时间内算对 → 算熟练度往上加。  
- **算对了但超时（卡顿）** → 系统当作「还差一点」：熟练度会往下扣一点，这道题之后更容易再出现。  
- 算错 → 熟练度降得更多，并进错题流程。

所以：**慢，也会被当成需要再练。**

### 3. 又快又对几次 → 少练（「冷宫」）

同一道具体算式（比如 `7×8`），若连续多次都在时限内做对，系统会认为已经掌握，**日常练习里会很少再出**。

但不会永远消失：隔一段时间会**少量抽查**（尤其是当初勉强才过关、练得不多的题），防止忘光。

## 错题本是干什么的

- 做错的题会进 **错题本**。  
- 在错题本或日常练习里，同一道题**连续做对够次数**（一般是 3 次）后，会从「待掌握」里拿掉。  
- 拿掉之后，日常卷也不会再当弱项狂推（和「已经掌握」对齐）。

若你觉得「错题本里删了，怎么每天还在出」——按新规则不应再出现；若仍出现，多半是又做错了，或还没连对够次数。

## 常见问题

### Q：必须开「限时答题」吗？

**不必。**  
开了：可编辑各题型目标时间。  
关了：不编辑目标时间，但每场开始前仍可选严格/加成模式；宽松模式下按需显示倒计时，系统仍按目标时间判断快慢。  
想练「又对又快」，建议打开目标时间，并选严格或适当加成。

### Q：三种计时模式有什么区别？

见上文「计时模式与补练规则」。简要：宽松不强制交卷、无星星加成；严格到点交卷、星星 +20%；自定义加成在目标时间上多给几秒缓冲，星星加成随秒数递减（每多 1 秒少 5 个百分点，最低无加成）。

### Q：补练会无限做下去吗？

不会。每场补练最多 `max(3, 总题数×15%)` 道，每道补练只一次机会，做完就结束。

### Q：为什么老出同一类题？

多半是这类题正确率低、或经常超时。系统在往弱处倾斜。  
若某类已经很熟，可在设置里少勾、或换自动分配让系统少给。

### Q：为什么有的题好久不出了？

可能已经进入「掌握」：又快又对够次数了。  
偶尔还会抽查；若抽查又慢或又错，会重新多练。

### Q：刚开始练一种新题型，感觉很随机？

对。新题型刚开始时，系统要先「见见世面」、多记一些做过的题，之后才会更明显地「专攻弱点」。这是正常现象。

### Q：星星和奖券？

做对（尤其限时内做对、连击）会得星星；可在 **我的奖券** 里兑换。  
这是鼓励，**不代替**「会不会」的判断。

### Q：沉浸模式是什么？

设置里可开：提交后几乎不打断、直接下一题；错题仍会记、本场末尾仍会补练。适合想少看对错动画、专心刷量时用。

### Q：「数字键盘答对即过」是什么？

设置里可开（默认开）：整数/小数题用数字键盘时，输入已与正确答案完全一致则自动提交，无需点确认；答错仍须确认。竖式、分数、余数题不受影响。

### Q：报告页有什么用？

**练习报告** 可看最近练得怎样、哪类偏弱、哪些刚掌握。适合家长隔几天看一眼，再决定设置里勾哪些题型。

## 给家长的使用建议

| 阶段 | 建议 |
|------|------|
| 入门 | 少勾几种题型，题量 20 左右；先求正确，再开限时 |
| 巩固九九表 | 勾 2～9 乘除相关；让系统帮着「没见过的先练」 |
| 提速 | 打开限时，按建议秒数设；关注「对但慢」是否变少 |
| 查漏 | 看错题本 + 报告；必要时用错题本「专项练」 |

---

## 和「数学课练习」有什么不同

| | 口算 | 数学课 / 题海 |
|--|------|----------------|
| 内容 | 按运算类型随机出算式 | 课本讲次里的固定题目 |
| 目标 | 又对又快（自动化） | 理解题意、掌握题型 |
| 进度 | 按每道算式的熟练度 | 按题目是否做过 / 掌握 |

两者可以一起用：口算练反应，数学练思路。
