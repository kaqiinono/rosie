# 鸡兔同笼问题解题方法
## 假设法
### 假设法条件
题目中有明确的或者隐藏的头和和腿和。隐藏的腿和一般是可以通过计算得到头和或者腿和。
* 头和：表示两种物品的总量，比如鸡和兔的总只数，雨天晴天的总天数，表示不同物品总体数量为头和
* 腿和：表示物品某个部位或者属性的总和，通常比头和大，不会比头和小，比如鸡和兔腿的总数，雨天和晴天种树的数量

### 假设法步骤
  * 第一步：假设全是****，验证对不对
  * 第二步：多/少：**
  * 第三步：换一次试试：↑↓ **
  * 第四步：换几次：**次
  * final：答题

## 分组法
在不知道头和或者腿和的情况下，用分组法
关键1：怎么分组
关键2：分几组

* 需要有明确的步骤图(?是需要用户输入的地方)
