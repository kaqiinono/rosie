- 难度：3 星

**题目**

一个等差数列 **2、5、8、11**、……请问，这个数列的第 **101** 项是多少？

**问题**：第 101 项是多少？（）

**解析**

- 公差 d = 5 − 2 = 3
- 第 n 项 = 首项 + (n − 1) × 公差
- 第 101 项 = 2 + (101 − 1) × 3 = 2 + 300 = **302**

**答案**：302
